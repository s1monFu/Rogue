### SAN
玩家操控的角色拥有SAN值，与怪物接触会缓慢减少SAN值，SAN值过低会导致进入疯狂形态，随机进入一种惩罚状态，无法使用正常科技，解锁部分图鉴（只限于疯狂状态使用），同时加概率暴击。开特定的门/做梦（回血）探索特定道路等都有可能得到奖励，但同时可能掉SAN。
